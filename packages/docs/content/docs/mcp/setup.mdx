## Quick Setup

Create an API key from **Dashboard &rarr; Keys**, then run this command:

```bash
claude mcp add --transport http \
  --header "Authorization: Bearer jrmy_your_key_here" \
  jeremy https://jeremy.khuur.dev/api/mcp
```

Nothing to install — Jeremy's MCP server runs entirely in the cloud.

## Manual Configuration

You can also add Jeremy directly to your Claude Code settings file (`.claude/settings.json` or `~/.claude/settings.json`):

```json
{
  "mcpServers": {
    "jeremy": {
      "type": "http",
      "url": "https://jeremy.khuur.dev/api/mcp",
      "headers": {
        "Authorization": "Bearer jrmy_your-api-key"
      }
    }
  }
}
```

## Verify

Once configured, restart Claude Code. The assistant should now have access to the `resolve-library-id` and `query-docs` tools. You can verify by asking Claude to look up documentation for a library you've added.

## Other MCP Clients

Jeremy's MCP endpoint uses the Streamable HTTP transport at `https://jeremy.khuur.dev/api/mcp`. Any MCP-compatible client that supports HTTP transport can connect by sending JSON-RPC 2.0 requests with an `Authorization: Bearer jrmy_...` header.
